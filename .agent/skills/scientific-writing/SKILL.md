# Scientific Writing

## IMRaD Structure
```
I — Introduction     (Why did you do this? What is the problem?)
M — Methods          (How did you do it?)
R — Results          (What did you find?)
aD — Discussion      (What does it mean? How does it compare?)
```

## Abstract Template (~250 words)
```
[Kontext/Motivation — 1-2 Sätze]
Diese Arbeit untersucht [Thema] im Kontext von [Rahmen].

[Forschungslücke — 1 Satz]
Bisherige Forschung hat [Lücke] nicht ausreichend adressiert.

[Zielsetzung — 1 Satz]
Ziel dieser Arbeit ist es, [Ziel] zu [erreichen/untersuchen/entwickeln].

[Methode — 2-3 Sätze]
Hierzu wird [Methode] angewandt. Es werden [Datenquellen] verwendet.

[Ergebnis — 2-3 Sätze]
Die Ergebnisse zeigen, dass [Hauptbefund 1]. Darüber hinaus wurde festgestellt, dass [Befund 2].

[Schlussfolgerung — 1-2 Sätze]
Die Arbeit leistet einen Beitrag zur [Disziplin] und zeigt, dass [Implikation].
```

## Argumentation Pattern
```
Behauptung:  "Multi-Agenten-Systeme verbessern die Effizienz."
Begründung:  "Durch Parallelisierung können Aufgaben gleichzeitig bearbeitet werden."
Beleg:       "(vgl. Müller et al., 2024, S. 45)"
Einschränkung: "Dies gilt jedoch nur, wenn die Kommunikationskosten gering sind."
```

## Linking Phrases (Deutsch)
```
Kausal:     "Daraus folgt...", "Infolgedessen...", "Daher..."
Kontrast:   "Jedoch...", "Im Gegensatz dazu...", "Wohingegen..."
Ergänzung:  "Darüber hinaus...", "Ferner...", "Zudem..."
Schluss:    "Zusammenfassend...", "Abschließend...", "Festzuhalten ist..."
```

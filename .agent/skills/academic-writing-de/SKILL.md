# German Academic Writing

## Thesis Structure (Masterarbeit)
```
1. Titelblatt
2. Abstract (DE + EN)
3. Inhaltsverzeichnis
4. Abkürzungsverzeichnis (if needed)
5. Einleitung (Introduction)
   5.1 Problemstellung (Problem statement)
   5.2 Zielsetzung (Objectives)
   5.3 Aufbau der Arbeit (Structure)
6. Theoretischer Hintergrund
7. Methodologie
8. Ergebnisse (Results)
9. Diskussion
10. Fazit und Ausblick (Conclusion & Outlook)
11. Literaturverzeichnis
12. Anhang (Appendices)
```

## Formal German Style Rules
- Use **passive voice** for objectivity: *"Es wurde untersucht..."* not *"Ich habe untersucht..."*
- **Nominalization**: *"die Durchführung"* preferred over *"durchführen"*
- Avoid contractions, colloquialisms, and personal pronouns where possible
- Use present tense for general truths, past for methods/results
- **Hedging**: *"Es kann festgestellt werden..."*, *"Aus den Ergebnissen lässt sich ableiten..."*

## Citation Formats

### APA (common in German universities)
```
In-text: (Autor, Jahr, S. Seitenzahl)
Bibliographie: Autor, A. (Jahr). Titel. Verlag.
```

### Chicago (footnotes)
```
¹ Autor, Titel (Ort: Verlag, Jahr), S. X.
```

## Useful Phrases
```
Einleitend lässt sich festhalten, dass...   (To introduce...)
Im Folgenden wird dargestellt...            (The following will show...)
Zusammenfassend kann gesagt werden...       (In summary...)
Es ist zu beachten, dass...                 (It should be noted...)
Vor dem Hintergrund dieser Erkenntnisse... (Against this backdrop...)
```
